
# React Pagedown

## Building/Developing

```sh
# Install and build component
npm ci
npm run build

# Build and serve demo
npm run dev
npm run serve

# Browse to localhost:3000
```

## Component Props

| Property  | Type   |
|-----------|--------|
| className | `string` |
| disabled | `boolean` |
| name | `string` |
| defaultValue | `string` |
| onUpdate | `(value: string) => void` |
| converter | `showdown.Converter` |
